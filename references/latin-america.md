# 拉丁美洲签证参考

## 核心策略：美签是关键

拉美国家大量依赖美签/申根签来裁定中国护照的入境资格。**如果你有有效美签，拉美一大半国家免签。**

## 重点国家

### 墨西哥
- **不需要美签就能免签的条件**：有有效美/加/日/英/申根签任一即可
- 这是中国护照去墨西哥的最常用方式
- 没有美签的话几乎必须办墨西哥签证（审核很严）

### 巴西
- 经常传出对中国免签，但政策反复变动
- **必须通过 WebSearch 确认当前最新政策**
- 如需签证：在线申请，材料要求多，出签约10-15工作日

### 阿根廷
- 凭美签/申根签 → 在线申请 AVE（电子旅行许可）
- AVE费用约 50美元
- 审核1-2周
- 没有美签 → 必须办贴纸签证，材料复杂

### 古巴
- 免签90天（中国和古巴互免）
- 但需要：旅游卡（Tourist Card，约20-30美元）
- 航空公司柜台或旅行社可代办
- 如果是去美国转机去古巴，注意美国对古巴的限制

## 环游拉美策略

```
最优路径：
持美签 → 墨西哥(免签) → 伯利兹(美签免签) → 危地马拉(免签)
→ 哥斯达黎加(美签免签) → 巴拿马(美签免签) 
→ 哥伦比亚(美签免签) → 秘鲁(美签免签) → 智利(美签免签)
→ 阿根廷(美签+AVE) → 巴西(需核实)

如果没有美签：
→ 先办美签，再走拉美
→ 或只去厄瓜多尔、苏里南等互免国家
```

## 需要核实的信息

- 巴西是否对中国免签（持续关注）
- 阿根廷AVE费用和等待时间
- 各国凭美签免签是否有停留天数变化
- 古巴旅游卡最新办理方式
