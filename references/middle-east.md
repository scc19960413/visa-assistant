# 中东地区签证参考

## 总览

中东国家对中国护照的签证政策差异极大，从互免到严格签证都有。

| 国家 | 政策 | 说明 |
|------|------|------|
| 阿联酋 | 互免30天 | 直接入境，可延期 |
| 卡塔尔 | 互免30天 | 直接入境 |
| 沙特阿拉伯 | 电子签/落地签 | 旅游开放，在线申请 |
| 阿曼 | 电子签/落地签 | 简化流程 |
| 巴林 | 电子签/落地签 | 较简单 |
| 科威特 | 电子签/落地签 | 需核实 |
| 约旦 | 落地签 | Jordan Pass可免签证费 |
| 黎巴嫩 | 落地签 | 免费 |
| 以色列 | 需签证 | 需提前在使领馆办理 |
| 伊朗 | 单方面免签21天 | 无需手续 |
| 土耳其 | 电子签（分分钟） | 在线填表付款即可 |
| 埃及 | 落地签25美元 | 南西奈省免签15天 |

## 以色列签证（需提前办理）

### 特点
- 需通过签证中心递交
- 审核严格，可能被要求补充材料
- 护照上有以色列签证/入境章 → 可能被一些阿拉伯国家拒绝入境

### 材料要点
- 在职证明（英文）
- 银行流水
- 详细英文行程
- 酒店预订单
- 往返机票
- 如有以色列方邀请更好

### 注意事项
- 如果后续要去伊朗/黎巴嫩/叙利亚等 → 建议先去这些国家再去以色列
- 以色列现在通常不盖章，改用入境小卡片

## 沙特阿拉伯（电子签）

- 2019年起对全球开放旅游
- 电子签在线申请：visitsaudi.com
- 费用约 440 沙特里亚尔（含保险）
- 1年多次，每次90天
- 出签快（通常1-2天）

## 土耳其（电子签）

- 在线申请：evisa.gov.tr
- 费用约 60美元
- 多次入境，180天内最多90天
- 通常几分钟出签
- 打印带着即可

- 以色列签证最新流程
