# 申根签证参考

## 基本信息

- 适用国家：27个申根成员国（法国、德国、意大利、西班牙、瑞士、荷兰、希腊等）
- 核心原则：申请**主要目的地国家**的签证；如果多国停留时间相同，申请**首入境国家**
- 常见类型：短期旅游/商务（C类，90天内）
- 指纹要求：2015年起需录指纹，有效期5年
- 官方渠道：各国在华使领馆 + 签证中心（VFS/TLS/BLS 等外包服务商）

## 材料清单结构

### 基础材料
- 护照（有效期 > 回国后3个月，至少2页空白）
- 申根签证申请表（各国版本略有差异）
- 35x45mm 白底近照2张
- 往返机票预订单（建议可退改的）
- 酒店预订单（覆盖全部行程）
- 旅行医疗保险（保额 ≥ 3万欧元/30万人民币）
- 行程计划（每一天的详细安排）

### 身份/职业证明
- 在职人员：在职证明（中英文、盖公章、含薪资职位准假信息）+ 营业执照副本盖章
- 自由职业：收入来源说明信 + 银行流水 + 辅助资产
- 学生：在校证明 + 学生证 + 父母担保材料
- 退休人员：退休证 + 退休金银行流水

### 财力证明
- 近6个月工资卡银行流水（余额建议 ≥ 3-5万，不用临时大额转入）
- 房产证、车产证（辅助加分）
- 存款证明（可选，非必须）

### 特殊人群
- 未成年人：出生证明公证+认证、父母同意出行公证+认证、在校证明
- 探亲访友：邀请函、邀请人身份证明、关系证明

## 领区划分

各大使馆领区划分不同，常见：
- 北京领区：华北、东北、西北
- 上海领区：江浙沪皖
- 广州领区：广东、广西、福建、海南
- 成都/重庆领区：西南地区
- 沈阳/武汉/西安等：视具体国家

具体以该国使馆官网为准。

## 办理流程

1. 确定申请国家 → 2. 在线填写申请表 → 3. 预约递交日期（旺季需提前1-2月）→ 4. 准备材料 → 5. 签证中心递交+录指纹 → 6. 等待审核（5-15工作日，旺季更长）→ 7. 出签/领取

## 中国申请人注意事项

- 行程单和机酒单是审核心核心，必须真实合理
- 银行流水不建议最后一次性大额转入，会被质疑
- 白本护照 + 单身 + 无房 → 拒签风险较高，建议先刷日韩/东南亚记录
- 不同申根国的拒签率差异很大：法国、意大利相对友好；德国、瑞士较严
- 首次申请不一定要申根签有效期很长，拿到后使用记录好，第二次申请会有优待

## 常见拒签原因

1. 行程不合理（每天安排太少/太多/地理跳跃）
2. 财力不足或流水异常
3. 回国约束力不足（单身、无房、无固定工作）
4. 材料不一致（在职证明和银行流水匹配不上）
5. 过往签证使用记录差

## 需要核实的信息（每次使用时通过 WebSearch 获取最新版）

- 各国签证费（2025-2026年价格）
- 预约等待时间（旺季/淡季）
- 各国特殊要求（如法国要求行程单用特定格式）
- 是否有电子签或简化流程
- 申根签证最新政策变化
