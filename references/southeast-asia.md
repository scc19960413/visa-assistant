# 东南亚签证参考

## 总览

东南亚是中国公民出境游最方便的区域，大部分国家对中国护照友好，签证政策简单。

| 国家 | 政策类型 | 说明 |
|------|----------|------|
| 泰国 | 免签 | 2024年起永久免签，单次停留30天 |
| 新加坡 | 免签 | 2024年起免签，单次停留30天 |
| 马来西亚 | 免签 | 2024年起免签，单次停留30天（至2026年底，后续需核实）|
| 越南 | 电子签/落地签 | 电子签90天多次，需提前在线申请 |
| 印度尼西亚 | 落地签/电子签 | 落地签30天可延一次，电子签更推荐 |
| 菲律宾 | 另纸签证 | 需通过旅行社办理，持有美/加/日/申根签可免签7天 |
| 柬埔寨 | 电子签/落地签 | 电子签推荐，落地签可能排队很久 |
| 老挝 | 落地签/电子签 | 手续简单，费用低 |
| 缅甸 | 电子签 | 需提前在线申请，旅游签28天 |
| 文莱 | 落地签 | 14天 |

## 通用注意事项

### 免签 ≠ 一定能入境
即使对中国公民免签，入境时仍可能被要求出示：
- 回程/续程机票
- 酒店预订
- 足够旅费现金（各国标准不同）
- 建议：即使免签，也随身带上行程单和酒店订单

### 护照有效期
绝大多数东南亚国家要求护照有效期 > 6个月。出发前务必检查。

### 落地签常见坑
- 只收现金（美元或当地货币）
- 可能被索要小费（泰国、越南、柬埔寨等偶有发生）
- 落地签柜台排长队（旺季），电子签通常更快

### 电子签
- 注意辨别官方网站（有很多山寨网站收费翻倍）
- 提前打印好电子签证页
- 电子签审核时间：通常1-3个工作日

## 东南亚作为"刷出境记录"的入门选择

对于白本护照、首次出国的申请人，东南亚是积累出境记录的最佳起点：
- 签证门槛低甚至免签
- 花费不高
- 有良好的东南亚出入境记录，对后续申请日韩、欧美签证有正面帮助

建议路线：东南亚 → 日韩 → 申根/澳新 → 美加

## 需要核实的信息（每次使用时通过 WebSearch 获取最新版）

- 各国免签/落地签政策的有效期和是否延续
- 新加坡、马来西亚免签政策是否有变化
- 各国落地签费用
- 缅甸内部局势（是否影响旅行安全）
- 菲律宾对中国公民的最新签证政策（波动较大）
