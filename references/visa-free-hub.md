# 中国护照签证类型全球速查

> ⚠️ 政策频繁变动，此文件为快速参考框架。具体使用时必须通过 WebSearch 核实最新政策。

---

## 使用逻辑

当用户询问任意国家/地区的签证要求时，按以下顺序判断：

```
用户询问「去XX要签证吗？」
  │
  ├─ 1. 查下表 → 互免/单方免签？ → ✅ 「不需要签证，可直接入境X天」
  │
  ├─ 2. 查下表 → 落地签？ → ✅ 「可落地签，带上护照+照片+现金即可」
  │
  ├─ 3. 查下表 → 电子签？ → ✅ 「在线申请电子签，X个工作日出签」
  │
  ├─ 4. 查下表 → 有条件免签？ → ⚠️ 「持有XX签证可免签入境」
  │
  └─ 5. 都不在 → 需提前办理签证 → 进入标准签证工作流
```
